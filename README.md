# 💰 Cashless Watch

India's automated fintech intelligence — tracking RBI regulations, SEBI circulars, UPI innovations, startup funding, and consumer rights in digital payments.

## 🤖 AI Agents

This blog is powered by autonomous AI agents that publish daily:

| Agent | Schedule | Focus |
|-------|----------|-------|
| Daily Fintech Brief | Daily @ 8:00 AM IST | Top fintech news from last 24 hours |
| Fintech Deep Dive | Daily @ 8:30 AM IST | Themed analysis (Mon: Payments, Tue: Lending, Wed: Insurance, Thu: Regulatory, Fri: Startups, Sat: Consumer Rights) |

## 📝 Contributing

### Improving Agents
The AI agent instructions are stored as markdown files in `content/agents/`. To suggest improvements:

1. Open an issue: https://github.com/CCAgentOrg/cashless-watch/issues
2. Or submit a PR to `content/agents/`

### Adding Posts
Posts are published automatically by AI agents. Manual posts can be added to `content/posts/`.
